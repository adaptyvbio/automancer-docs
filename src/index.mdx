# {Brand}

{Brand} is a software application which makes it possible to automate lab protocols.


## Features

- Optional graphical user interface – The application can be fully controlled from a graphical user interface.
- Ease of use – Protocols are defined in a Yaml-like language which can be learned in less than an hour.
- Modular design – Support for a new device in your setup can be added by simply installing a Python package. If no corresponding package is available, you can develop yours and have it seamlessly integrate with the user interface.
- Reliability and reproducibility – {Brand} has support for error recovery, real-time notifications and log recording, which can be used to determine where an experiment went wrong.
- Remote connection – Experiments can be controlled remotely, either through the graphical user interface or with a JSON-over-TCP endpoint.
- Platform support – Linux, macOS, Windows
